# PWA Icons

Generate icons at https://realfavicongenerator.net/ and place here:

Required sizes:
- icon-72.png
- icon-96.png  
- icon-128.png
- icon-144.png
- icon-152.png
- icon-192.png (required for PWA)
- icon-384.png
- icon-512.png (required for PWA)

Also create:
- default-album.png (for albums without cover art)
- misc-cover.png (for Misc collection)
- og-image.png (for social sharing, 1200x630)
